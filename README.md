# EffortX

> AI-powered GitHub contribution verification on Solana.
>
> Turning real engineering work into verifiable on-chain proof, developer reputation, and AI utility.

---

## 🚀 Overview

EffortX is an AI-powered developer reputation platform built on Solana.

Instead of measuring developers only through commit counts, streaks, or resumes, EffortX analyzes the actual quality and impact of GitHub contributions.

Developers can:

* Submit GitHub commits or pull requests
* Get AI-based contribution analysis
* Receive effort scores and engineering insights
* Store contribution proofs on-chain
* Earn ECOIN rewards
* Use ECOIN inside the AI Playground as AI credits

EffortX transforms GitHub activity into:

* Verifiable reputation
* On-chain proof
* AI utility
* Developer ecosystem identity

---

# ❗ Problem

Current developer evaluation systems are broken.

GitHub mainly measures activity:

* commit counts
* contribution streaks
* lines changed

LinkedIn mainly measures claims:

* resumes
* self-reported achievements
* endorsements

But neither platform measures actual engineering contribution quality.

A production-critical architecture fix and a typo correction can both appear as "1 commit".

This problem becomes even bigger in the AI era where generating large amounts of code is becoming easier.

EffortX solves this by creating an AI-powered qualitative reputation layer for developers.
